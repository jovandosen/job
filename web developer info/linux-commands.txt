21. Recursively set read, write, and execute permissions to the file owner 
	and no permissions for all other users on a given directory

sudo chmod -R 700 dirname
//////////////////////////////////////////////////////////////////////////////////
22. For example, to change the permissions of all files and subdirectories 
	under the /var/www directory to 755 you would use:

sudo chmod -R 755 /var/www
//////////////////////////////////////////////////////////////////////////////////
23. The most common scenario is to recursively change the website 
	file’s permissions to 644 and directory’s permissions to 755

	-> Using the numeric method:

		find /var/www/my_website -type d -exec chmod 755 {} \;
		find /var/www/my_website -type f -exec chmod 644 {} \;

	-> Using the symbolic method:

		find /var/www/my_website -type d -exec chmod u=rwx,go=rx {} \;
		find /var/www/my_website -type f -exec chmod u=rw,go=r {} \;
